# Olá, seja bem-vindo ao meu perfil! 👋

**Fullstack Developer com foco em Frontend**

Desenvolvedor de software com foco em frontend, atuando principalmente com **React**, **Next.js** e **TypeScript** na construção de produtos SaaS. Tenho experiência em arquitetura de interfaces, formulários complexos (React Hook Form + Zod), integração com APIs REST e GraphQL, gerenciamento de estado global (Zustand e Context API) e criação de experiências responsivas e escaláveis (Flexbox, CSS Grid, Tailwind CSS).

Também atuo no backend com **Node.js** e **NestJS**, especialmente na criação de APIs, regras de negócio, validações com class-validator/class-transformer e persistência de dados com **Prisma** e **PostgreSQL**. No mobile, desenvolvo aplicações multiplataforma com **React Native (Expo)**.

- 🔭 Atualmente trabalhando com SaaS de pesquisas quantitativas e qualitativas
- 🌱 Estudando Tanstack Start e aprofundando em testes com Jest e Testing Library
- 📫 Contato: **icaroregisalmeida@gmail.com**
- 🌐 Portfólio: [icaroregis.github.io](https://icaroregis.github.io/)
